* Docker :noexport_1:
** Copy container from machine A to machine B
*** Method 1
Save docker container as tar file.
#+begin_src shell
docker export $(docker images | grep itorafael/ofa | awk '{print $3}') > /home/rafael.ito/ct_example.tar
#+end_src

Import in another machine.
#+begin_src shell
cat /home/rafael.ito/ct_example.tar | docker import - itorafael/ofa:jupyter
#+end_src
*** Method 2
#+begin_src shell
docker save -o /home/rafael.ito/ct_example.tar itorafael/ofa:jupyter
docker load -i /home/rafael.ito/ct_example.tar
#+end_src

** Magit
*** Setting a GitHub repository
Create a directory. Then =M-x magit-status= and type =y= for "Initialize a new git repository?".
Stage and commit normally.
Create a git repo from the CLI:
#+begin_src shell
gh repo create <MyAwesomeRepo> --public --description "Description of my awesome repo"
#+end_src
Open the =Remote= popup by pressing =M=; Add a remote with =a=; Name it =origin=; Paste the URL of the git repo (eg: =git@github.com:username/MyAwesomeRepo.git=); Accept =remote.pushDefault= as =origin=.
Push to remote normally.

** Accelerate audio & video
*** 0.625
Note that (1 / 0.625) = 1.6
#+begin_src shell
ffmpeg -i input.mp4 -filter_complex "[0:v]setpts=0.625*PTS[v];[0:a]atempo=1.6[a]" -map "[v]" -map "[a]" output.mp4
#+end_src
*** 0.9
Note that (1 / 0.9) = 1.111111111
#+begin_src shell
ffmpeg -i input.mp4 -filter_complex "[0:v]setpts=0.9*PTS[v];[0:a]atempo=1.111111111[a]" -map "[v]" -map "[a]" output.mp4
#+end_src
*** 0.8
Note that (1 / 0.8) = 1.25
#+begin_src shell
ffmpeg -i input.mp4 -filter_complex "[0:v]setpts=0.8*PTS[v];[0:a]atempo=1.25[a]" -map "[v]" -map "[a]" output.mp4
#+end_src

* Scribd :noexport_1:
** How to view a document
Take the URL for the document:
https://www.scribd.com/document/123456789/my-awesome-pdf-document

Replace =document= with =embeds=, and the title (=my-awesome-pdf-document=, in this case) with =content=:
https://www.scribd.com/embeds/546052355/content
** How to download a file
Go to Scribd downloader website. For example:
- https://scribd.vdownloaders.com/
- https://docdownloader.com/
- https://vpdfs.com/
- https://scribd.vpdfs.com/
* SSH :noexport_1:
** Generating a new SSH key
*** RSA
#+begin_src shell
ssh-keygen -t rsa -b 4096 -C "comment-name-or-email"
#+end_src
*** ED25519
#+begin_src shell
ssh-keygen -t ed25519 -C "comment-name-or-email"
#+end_src
** ssh-agent
*** Adding your SSH key to the ssh-agent
#+begin_src shell
ssh-add ~/.ssh/id_rsa
ssh-add ~/.ssh/ed25519
#+end_src
*** Lists the keys managed by ssh-agent
#+begin_src shell
ssh-add -l
#+end_src
*** Start ssh-agent
#+begin_src shell
eval "$(ssh-agent -s)"
#+end_src
