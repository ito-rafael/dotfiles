* Information
** How to load config file
1. Copy the config file into your $XDG_CONFIG_HOME ($HOME/.config).
#+begin_src sh :tangle no
mkdir -p $XDG_CONFIG_HOME/surfingkeys
cp ./surfingkeys.js $XDG_CONFIG_HOME/surfingkeys.js
#+end_src

1. Open the SurfingKeys settings page: =chrome-extension://gfbliohnnapiefjpjlpjnehglfpaknnc/pages/options.html=

2. Change the =Load settings from= field and insert the *full* path for your config file: =file:///home/<USERNAME>/.config/surfingkeys/surfingkeys.js=

** Unbind
*** vim-like marks
**** "m": "Add current URL to vim-like marks"
#+begin_src js
api.unmap('m');
#+end_src
*** Mouse Click
**** "af": "Open a link in active new tab"
#+begin_src js
api.unmap('af');
#+end_src
*** Omnibar
**** "ab": "Bookmark current page to selected folder"
#+begin_src js
api.unmap('ab');
#+end_src
* Normal mode
** Scroll
#+begin_src js
api.mapkey("n", "Scroll left",  () => { api.Normal.scroll("left"); });
api.mapkey("e", "Scroll down",  () => { api.Normal.scroll("down"); });
api.mapkey("i", "Scroll up",    () => { api.Normal.scroll("up"); });
api.mapkey("o", "Scroll right", () => { api.Normal.scroll("right"); });
#+end_src
*** Backup
Save the original bindings into aliases to avoids dependency/order issues.
#+begin_src js
api.vmap("_h",  "h");
api.vmap("_j",  "j");
api.vmap("_k",  "k");
api.vmap("_l",  "l");
#+end_src
*** Unmap
Unmap the orinal bindings that were backed up.
#+begin_src js
api.vunmap("h");
api.vunmap("j");
api.vunmap("k");
api.vunmap("l");
#+end_src
